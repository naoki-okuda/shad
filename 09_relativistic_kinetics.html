09 relativistic kinetics

おっしゃる通り、相対論的流体力学や相対論的熱力学は、まさに「物質バルクの平均的な静止座標系」を各時空の点（流体要素）ごとに定義し、「その座標系において、我々が知っている非相対論的な熱力学の法則が（ほぼ）そのまま成立する」と仮定することで構築されています。

このアプローチの核心は、ご指摘の2つの概念にあります。

1. 🔑 鍵となる「局所静止系」 (Local Rest Frame)

  • 「時間が排除されている」というご指摘は、的を射ています。熱力学は「状態A」から「状態B」への変化を扱いますが、その間の「時間」には（基本的には）立ち入りません。
  • 流体全体が複雑に運動（膨張、収縮、剪断）していても、ごく微小な「流体要素」一つだけを取り出して、それと一緒に動く座標系を考えます。これが「局所流体静止系 (Local Fluid Rest Frame, LRF)」です。
  • この座標系では、その流体要素の平均3次元運動量はゼロです。粒子は個々に熱運動していますが、「バルク（塊）」としては静止しています。
  • このLRFは、各時空の点ごとに一意に（あるいは特定の定義、例えばエネルギー流がゼロになる系＝Landau-Lifshitzフレーム、粒子流がゼロになる系＝Eckartフレームとして）指定されます。

2. 🌍 局所熱平衡 (Local Thermodynamic Equilibrium, LTE) の仮定

  • これが物理的に最も強い「束縛」あるいは「仮定」です。
  • 我々は、「各流体要素は、その局所静止系 (LRF) において、熱力学的な平衡状態にある」と仮定します。これが局所熱平衡 (LTE) です。
  • 全体としては（例えば爆発現象のように）全く平衡ではありませんが、ミクロな衝突時間がマクロな現象の時間スケールよりずっと短ければ、各要素は「局所的に」平衡に達する、という考え方です。
  • Jüttner分布は、まさにこのLTE状態を記述する分布関数です。

3. 📖 熱力学関数の構築

ご質問の「この慣性系上での完全な熱力学関数が作られるということ？」について。

まさしく、その通りです。

LTEを仮定したおかげで、我々はそのLRFにおいて、非相対論的な熱力学と全く同じ手続きで、すべての熱力学的な「状態量」を定義できます。

  • スカラ量: 圧力 \(P\)、数密度 \(n\)、内部エネルギー密度 \(\epsilon\)、エントロピー密度 \(s\)、温度 \(T\)
  • 第一法則: \(d\epsilon = Tds + \mu dn\) (または \(d\epsilon = Tds - P dV_{\text{specific}}\))
  • 第二法則: \(s\) が定義され、その生成率は正またはゼロである \(T \partial_\mu s^\mu \ge 0\)

これらの基本的な状態量 \(P, T, \epsilon, s\) がLRFで定義できたため、そこからヘルムホルツの自由エネルギー \(F\)、ギブズの自由エネルギー \(G\)、エンタルピー \(H\) といった、我々が知るすべての熱力学関数（ポテンシャル）を、LRF上で一意に構築することが可能になります。

まとめ

相対論と熱力学の「不思議」な融合は、以下のステップで行われます。

  1. 局所静止系 (LRF) の導入: 「時間」が問題にならない、系と一緒に動く「静止した」座標系を各点で見つける。
  2. 局所熱平衡 (LTE) の仮定: そのLRFでは、非相対論的な熱力学が成立する「平衡状態」にあると仮定する。
  3. 熱力学関数の定義: \(P, T, s, \epsilon\) などをLRFで定義し、第一・第二法則が成立することを要請する。（例： \(P=nT\) や \(\epsilon = \epsilon(T, n)\)）
  4. 座標変換: これらのLRFで定義された量（特にエネルギー・運動量テンソル \(T^{\mu\nu}\)）が、実験室系などの別の慣性系から見たらどう見えるかを、ローレンツ変換を使って記述する。

相対論的ボルツマン方程

あるいはプラズマなどでは衝突項を0とした相対論的ヴラソフ方程式

  1. ミクロな記述 (Kinetics)

    • 基本方程式: 相対論的ボルツマン方程式
    • 共変な形式で記述される。 $\(p^\mu \frac{\partial f}{\partial x^\mu} + F^\mu \frac{\partial f}{\partial p^\mu} = C[f, f]\)$ （\(p^\mu\) は4元運動量、\(C[f, f]\) は衝突項です）
    • 熱平衡解: この方程式の衝突項をゼロにする（\(C[f, f]=0\)）状態、つまり熱平衡状態の解が、まさしくJüttner分布 \(f_{\rm Jüttner}\) です。
    • これは、非相対論的なボルツマン方程式の平衡解がマクスウェル分布になるのと同じ関係です。

  2. マクロな記述 (Fluid Dynamics)

    • 導出方法: Kinetcsの方程式（ボルツマン方程式）のモーメント（運動量空間での積分）を取ることで、流体の方程式を導出します。
    • 0階モーメント: \(n = \int f d^3p\) \(\rightarrow\) 粒子数密度 \(n\) （と粒子数保存則）
    • 1階モーメント: \(T^{\mu\nu} = \int p^\mu p^\nu f d^3p\) \(\rightarrow\) エネルギー・運動量テンソル \(T^{\mu\nu}\) （とエネルギー・運動量保存則 \(\partial_\mu T^{\mu\nu} = 0\)）

おっしゃる通り、相対論的ボルツマン方程式がローレンツ共変な（座標変換に対して不変な）形で書けることは、相対論的な理論として必須の要件です。

ご質問の「他に束縛はないのか？」という点については、非相対論的なボルツマン方程式が持つ仮定（束縛）の多くを、相対論的な文脈でも引き継いでいます。

主なものは以下の3点です。

1. 分子カオスの仮定 (Stosszahlansatz)

これがボルツマン方程式の最も根本的な「束縛」（仮定）です。

  • 内容: 衝突する2つの粒子は、衝突直前には何の相関も持たないと仮定します。
  • 意味: これにより、2粒子分布関数 \(f_2(x_1, p_1, x_2, p_2)\) を、1粒子分布関数 \(f_1\) の積 \(f_1(x, p_1) f_1(x, p_2)\) で近似できます。
  • 結果: この仮定があるからこそ、複雑な多体問題が、1粒子分布関数 \(f\) だけで閉じた方程式（ボルツマン方程式）に帰着されます。逆に言えば、粒子間に強い相関がある場合（例：高密度な液体、強結合プラズマ）では、ボルツマン方程式は適用できません。

2. オンシェル (On-Mass-Shell) の束縛

  • 内容: 扱っている粒子は全て「オンシェル」である、つまりその4元運動量 \(p^\mu\) は、質量 \(m\) の関係式 \(p^\mu p_\mu = (mc)^2\) （または \(p^2 = m^2\)）を常に満たしている、と仮定します。
  • 意味: 分布関数 \(f(x, p)\) が定義される8次元の位相空間 \((x^\mu, p^\mu)\) のうち、運動量空間は \(p^2=m^2\) という曲面（マスシェル）上に束縛されています。
  • 結果: 不確定性原理によって生じるような、短時間だけエネルギー保存則を破る「オフシェル」の状態は考慮しません。

3. 古典的記述（量子効果の無視）

  • 内容: 標準的なボルツマン方程式は、粒子を古典的な点粒子として扱います。
  • 意味: 量子的な効果は無視されます。特に重要なのは以下の2つです。
    1. 量子統計性: フェルミ粒子の場合、衝突先のスロットが既に埋まっていれば衝突できない（パウリの排他律/Pauli Blocking）という効果が無視されます。ボース粒子の場合は衝突先が埋まっているとむしろ衝突しやすくなる（ボース増幅/Bose Enhancement）効果が無視されます。
    2. 不確定性原理: \(f(x, p)\) という「位置 \(x\) と運動量 \(p\) を同時に持つ粒子の数」を定義すること自体が、\(\Delta x \Delta p \gtrsim \hbar\) という制約を無視した古典的な近似です。

これらの束縛のうち、特に3番目の「量子統計性」は、高密度な系（縮退した電子ガス、中性子星など）を扱う場合には無視できません。

その場合は、衝突項 \(C[f, f]\) に \((1-f)\)（フェルミ粒子）や \((1+f)\)（ボース粒子）といった因子を導入して量子効果を取り入れた、相対論的ボルツマン・ウーリング・ウーレンベック (Relativistic Boltzmann-Uehling-Uhlenbeck, RBUU) 方程式（または単に量子ボルツマン方程式）へと拡張する必要があります。
