Wave action

2. なぜ Wave Action が「強力」なのか？

Wave Action が「強力」と呼ばれる最大の理由は、エネルギーが保存しない状況でもそれが保存する（あるいは保存則を満たす）からです。

エネルギー \(E\) vs Wave Action \(J\)

  1. エネルギー (\(E\)) は保存しない 例えば、波が平均流（プラズマの流れや海流など）からエネルギーをもらって成長する（不安定性）場合や、逆に波が平均流にエネルギーを渡す場合、波自体のエネルギー \(E\) は時間とともに変化します。つまり、保存しません。

  2. Wave Action (\(J\)) は保存する ところが、散逸（摩擦や抵抗）がない系では、波が平均流とエネルギーをやり取りしている最中でも、Wave Action \(J\) は保存します（正確には、Wave Action 密度 \(A\) とそのフラックス \(\boldsymbol{F}\) が \(\frac{\partial A}{\partial t} + \nabla \cdot \boldsymbol{F} = 0\) という保存則を満たします）。

🌍 具体例：勾配のある場所を進む波

海で、水深が浅くなる（ゆっくりと勾配がある）場所に向かって波が進むと、波の振幅は大きくなります（砕波）。

  • このとき、波のエネルギー \(E\) も周波数 \(\omega\) も変化してしまいます。
  • しかし、Wave Action \(J \approx E/\omega\) は（ゆっくりとした変化であれば）保存しようとします。
  • この「Wave Action が保存する」という条件を使うことで、エネルギーや周波数がどう変化し、結果として振幅がどれだけ大きくなるかを、非常に見通しよく計算できます。

これは、プラズマ中で密度や磁場の勾配がある場所を波が伝播する際にも全く同様に当てはまり、波の振幅変化（成長や減衰）を追跡する上で極めて強力な道具となります。非線形相互作用（3波相互作用など）の記述にも用いられます（Manley-Roweの関係など）。

3. ラグランジアン密度との関係

では、ラグランジアン密度 \(\mathcal{L}\) と Wave Action \(J\) はどう関係するのか。

ここで登場するのが「Whitham の平均化ラグランジアン法」です。

  1. 系のラグランジアン密度 \(\mathcal{L}\) から出発します。
  2. 波の速い振動（例：\(\cos(kx - \omega t)\)）と、振幅や波数がゆっくり変化する時空間スケールを分離します。
  3. \(\mathcal{L}\) を、この速い振動の1周期にわたって平均化したラグランジアン \(\langle \mathcal{L} \rangle\) を作ります。
  4. この \(\langle \mathcal{L} \rangle\) に対して変分原理（最小作用の原理）を適用します。
  5. すると、驚くべきことに、この操作から波の分散関係（\(\omega(k)\)）とWave Action の保存則が、統一的かつ系統的に導出されるのです。

つまり、Wave Action は、元のラグランジアン密度を「波のスケールで平均化したもの」から導かれる保存量、という関係にあります。

まとめ

「Wave Action を考えるのが強力」とは、

波のエネルギーが保存しないような複雑な状況（媒質が不均一、平均流がある、非線形相互作用がある）でも、Wave Action (\(J \propto E/\omega'\)) という量は保存則を満たす。

この保存則は、元の系のラグランジアン密度 \(\mathcal{L}\) を平均化するという系統的な手法（Whitham の方法）によって導出できる。

これにより、運動方程式を直接解くよりもはるかに見通し良く、波の振幅の振る舞いを解析できる。

ということを意味しています。

いいところ突いてる。結論から言うと

「考える。しかも体系的にやる流儀がちゃんとある」 です。

ただし、それはもう 「線形波動」でも「普通の流体方程式」でもなくて、 多重スケール展開・平均化・波動運動論の世界に入ります。

1. 何が起きているか（整理）

あなたの言っている構図はこう：

  • 0次：背景（平均場）
  • 1次：線形波動（振幅 ( \epsilon )）

  • 2次：

  • 波どうしの自己相互作用

  • 波が平均場に与える輸送（運動量・エネルギー）

重要なのは：

輸送は 1次を平均すると消えて、2次で初めて残る

これは $$ \langle \delta v \rangle = 0,\quad \langle \delta v,\delta v \rangle \neq 0 $$ という、ほぼ統計力学と同じ構造。

2. 「一次を取り除いた二次の方程式」は実際に作る

(A) Reynolds分解 → 波による応力項

流体量を $$ \mathbf{v} = \bar{\mathbf{v}} + \tilde{\mathbf{v}} $$

と分けて平均を取る。

すると運動方程式は

\[ \rho(\partial_t \bar{\mathbf{v}} + \bar{\mathbf{v}}\cdot\nabla\bar{\mathbf{v}}) = -\nabla \bar{p} ;\color{blue}{-\nabla\cdot\langle \rho,\tilde{\mathbf{v}}\tilde{\mathbf{v}}\rangle} \]

この青い項が

  • 波による運動量輸送
  • 放射応力（radiation stress）
  • ponderomotive force

これは「一次の波動を消して、二次の効果だけ残した平均方程式」。

(B) 多重スケール展開（WKB / multiple-scale）

小パラメータ $ \epsilon $ を入れて

$$ \begin{aligned} f &= f_0(X,T)

  • \epsilon f_1(x,t;X,T)

  • \epsilon^2 f_2 + \cdots \ X &= \epsilon x,\quad T=\epsilon t \end{aligned} $$

  • (f_1)：線形波動

  • (f_2)：平均場の進化

を入れて整理すると：

  • (O(\epsilon))：分散関係
  • (O(\epsilon^2))：共鳴条件・平均場へのバックリアクション

が出てくる。

このとき $$ \partial_T f_0 = -\nabla_X\cdot \langle f_1 f_1 \rangle $$ みたいな式が現れる。

(C) wave action 保存則（あなたが気にしてたやつ）

実はこれも二次まで取った結果を、うまくまとめたもの

$$ \partial_t \mathcal{A}

  • \nabla\cdot(\mathbf{v}_g \mathcal{A}) = 0 $$

  • \(\mathcal{A} \sim E/\omega\)

  • エネルギー (E) 自体が2次量

👉 「波を自由度として残し、平均場と結合させた閉じた理論」

3. じゃあ「二次だけの閉じた方程式」は書けるのか？

一般には無理。

理由：

  • 2次項は $$ \langle f_1 f_1 \rangle $$ を含む
  • これは 波の位相・スペクトルを知らないと決まらない

だから：

  • 平均場の方程式
  • 波動（action, energy, spectrum）の方程式

を 同時に解くしかない。

👉 これが wave–mean flow interaction 波動運動論 準線形理論（quasilinear theory） の正体。

4. 物理的な見方（直感）

線形波：

  • 「粒子は揺れるだけ」
  • 「平均移動しない」

二次効果：

  • 位相がずれる
  • 非対称が蓄積する
  • ゆっくりしたドリフトや流れが生まれる

例：

  • Stokes drift（表面波）
  • ゾーナルフロー生成
  • レーザーの ponderomotive force
  • プラズマ波による運動量輸送

5. あなたの疑問への短い答え

一次の摂動を取り除いた、二次の水準まで考えた方程式は？

ある。しかも重要。

ただし：

  • 単独では閉じない
  • 波動の自由度とセットで初めて意味を持つ

  • その整理された形が Reynolds応力 / wave action / quasilinear theory

  • 「なぜ線形波が必ず二次作用を持つのか」を 最小モデルで式から見せる

  • それとも レーザープラズマ or 音波減衰に即した具体例

波は「基本的には」媒質を運ばないが、 非線形・減衰・境界が入ると、必ず輸送に変換される。

その変換の仕方を整理すると見通しがよくなる。

1. 線形・理想波動：輸送するのは「場」

まずいちばん教科書的な状況。

  • 微小振幅
  • 一様媒質
  • 無限空間
  • 減衰なし

このとき波は

  • エネルギー
  • 運動量
  • （場合によって）波作用（wave action）

を運ぶけど、

  • 媒質のラグランジュ平均の位置は動かない
  • 質量流束は 0

たとえば音波： $$ \langle \rho u \rangle = 0 $$ （オイラー平均では）

この意味で

「波が立っても媒質は移動しない」 は一次理論では正しい。

2. でも「運動量を運ぶ」のは事実

ここが一段深いところ。

波は $$ \Pi \sim \langle \delta p, \delta u \rangle $$ みたいな 二次量として 運動量フラックスを持つ。

重要なのは：

  • 運動量は 保存量
  • どこかで波が消えれば（吸収・減衰・反射）
  • その運動量は 媒質に渡される

つまり：

波が運ぶ運動量は、必ずどこかで流れに変わる

3. 一次摂動が0次プロファイルに効く理由

君が言ってる

0次のプロファイルにも一次摂動の伝播は影響を受ける

これは完全に正しい。

理由は単純で、

  • 平均すると二次が残る
  • 二次は0次方程式に入る

典型例：

  • Reynolds応力
  • ponderomotive force
  • radiation pressure
  • wave-induced stress

流体方程式で言うと： $$ \partial_t U_0 = - \partial_x \langle u_1 u_1 \rangle + \cdots $$

一次は「揺れ」だけど、 揺れの相関が平均流を駆動する。

4. 海の波：なぜ質量輸送が見えるか

海の波は最高の例だね。

(1) Stokes drift

有限振幅波では：

  • 粒子の軌道が閉じない
  • 表層ほど前進が大きい
\[ u_{\rm drift} \sim \omega k a^2 e^{2kz} \]

これ、純粋に非線形効果。

線形理論では消えていたやつ。

(2) 波の減衰と砕波

波打ち際では：

  • 砕波 → エネルギー散逸
  • 波の運動量が失われる
  • 保存則により → 平均流が生成

これが

  • 海浜流
  • 沿岸流
  • セットアップ

6. プラズマの文脈に寄せると

たぶん君の頭にはこれがあるよね：

  • レーザー駆動波
  • プラズマ波
  • エネルギー輸送
  • 非熱電子

ここでは：

  • Langmuir波の減衰 → 電子加熱
  • 波の群速度 → エネルギーフラックス
  • 波圧 → 密度プロファイル変形

つまり：

波は「輸送そのもの」ではなく、 輸送へ変換される中間キャリア

7. 直感的なまとめ

いい比喩を使うなら：

波は「運ぶ意思」を持たないが、 壊れると必ず何かを運ぶ

揺れている間は情報と保存量を運び、 消える瞬間に物質世界に痕跡を残す。

もしよければ次は

  • wave action と輸送の関係
  • 波エネルギー密度はどれくらい大きくなれるか
  • 流れと波を分ける意味はどこまであるか

うん、その式は基本的に正しいです。ただし、成立条件と意味づけをかなり厳密に押さえておく必要があります。 Naoki さんが気にしている「波のエネルギー容量はどれくらいか」「流れの運動エネルギーとどう切り分けるか」という文脈に、まさに直結する式です。

結論（まず要点）

線形・弱減衰のプラズマ波に対して、 単色平面波 ( (\omega,\mathbf{k}) ) について

\[ \boxed{ W_w = \frac{|E|^2}{16\pi}, \frac{\partial}{\partial\omega} \bigl[\omega,\varepsilon(\omega,\mathbf{k})\bigr] } \]

という形で 波動エネルギー密度（正確には 平均エネルギー密度）が書けます。

  • 電磁場のエネルギー
  • 粒子の秩序運動エネルギー（振動エネルギー）

を まとめた量 です。

どこから来る式か（物理的意味）

この式は

  • 分散媒質中の電磁エネルギー保存則
  • または Poynting 定理を分散媒質に拡張したもの
  • Brillouin, Landau–Lifshitz, Stix などの古典的結果

に基づきます。

ポイントは：

プラズマ中では 「電場エネルギー = ( |E|^2/8\pi )」 だけでは 全エネルギーを数え切れない

ということ。

粒子が電場に引きずられて振動しており、その運動エネルギーが \(\varepsilon(\omega,k)\) の (\omega)-依存性として隠れている。

それを正しく拾うのが

\[ \frac{\partial}{\partial\omega}(\omega\varepsilon) \]

という因子。

係数 (16\pi) について

これは

  • 実効値（時間平均）
  • 複素振幅表記
  • ガウス単位系

を使った結果です。

例えば $$ \mathbf{E} = \Re{ \mathbf{E}_0 e^{-i\omega t}} $$ なら

\[ \langle |E|^2\rangle = \frac{1}{2}|E_0|^2 \]

が効いて、最終的に (16\pi) が出ます。 文献によって (8\pi) が出てきても、定義の違いなので注意。

どういう場合にダメか

この式は万能ではありません。

❌ 成立しない／危ないケース

  • 強い減衰（(\Im \varepsilon \not\ll \Re \varepsilon)）
  • 非線形波動
  • 波が粒子を捕捉している場合
  • 強い空間非一様性
  • 波束の急激な時間変化

特に Landau 減衰が強いときは、 「波のエネルギー密度」という概念自体が曖昧になります。

wave action との関係（Naoki さんの関心点）

このエネルギー密度から

\[ \boxed{ \mathcal{N} = \frac{W_w}{\omega} } \]

を定義したものが wave action。

  • 断熱変化に対して保存
  • 背景がゆっくり変わるときに有効
  • 幾何光学近似の本体

なので、

「エネルギーは保存しないが、action は保存する」

というあの話につながります。

比熱との比較という観点で言うと

Naoki さんの直感はほぼ正しくて、

  • 通常の線形波動では (W_w \ll nT)
  • 大きくなると それはもう「流れ」や「非線形運動」

として扱うべき領域に入る。

この式は 「波として数えられる最大のエネルギー」 を与えているとも言えます。

参考になる文献

  • Stix, Waves in Plasmas
  • Landau & Lifshitz, Electrodynamics of Continuous Media
  • Breizman & Ryutov (wave action)

もしよければ次に：

  • 電子プラズマ波で具体的に (\varepsilon(\omega)) を入れて評価する
  • 「このエネルギーのどれくらいが粒子側か？」を分解する

「一次摂動を解いて、振幅²をエネルギーに対応させる」だけでは不十分 ただし、正しい枠組みに乗せれば同値な結果に到達できる。

その「正しい枠組み」が何か、という話です。

1️⃣ なぜ「一次摂動 → 振幅²」だけではダメか

流体方程式（あるいは Vlasov）を

\[ f = f_0 + \delta f \]

として一次で解くと、 \(\delta n\)\(\delta u\), \(\delta E\) は確かに求まる。

エネルギーは二次量 → 本質的に 二次摂動 の情報が必要

典型的な落とし穴

例えば流体で書くと：

[ \mathcal{E} = \frac{1}{2}mn u^2 + \frac{p}{\gamma-1} + \frac{E^2+B^2}{8\pi} ]

これを展開すると：

[ \mathcal{E} = \mathcal{E}_0

  • (\text{一次})
  • (\text{二次})

  • \cdots ]

  • 一次項：平均すると 0

  • 二次項：これが「波のエネルギー」

👉 一次摂動だけ解いても、 どの二次の組み合わせが保存量になるかは自動では出てこない。

2️⃣ 正攻法①：エネルギー保存則から導出（王道）

一番きれいなのはこれ。

(a) 出発点

  • Maxwell 方程式
  • 流体 or Vlasov 方程式

から エネルギー保存則 を作る：

$$ \frac{\partial}{\partial t} \left( \frac{E^2+B^2}{8\pi}

  • \sum_s \int d^3v, \frac{mv^2}{2} f_s \right)
  • \nabla\cdot \mathbf{S} = 0 $$

（Poynting 定理）

(b) 線形化

\[ E = E_1,\quad f = f_0 + f_1 \]

として 二次まで残す。

(c) 時間平均

単色波 (e^{-i\omega t}) に対して時間平均すると

  • 一次項は消える
  • 二次項だけが残る

(d) 分散関係を使う

ここで $$ \varepsilon(\omega,k) $$ が自然に登場して、

\[ W_w = \frac{|E|^2}{16\pi} \frac{\partial}{\partial\omega} \left[ \omega \varepsilon(\omega,k) \right] \]

が出てくる。

👉 エネルギー密度は「保存則」からしか一意に定義できない これが重要。

3️⃣ 正攻法②：変分原理（wave action 直結）

これは少し抽象的だけど、本質が一番見える。

(a) 波のラグランジアン密度

線形波に対して

\[ \mathcal{L} = \frac{1}{16\pi} \left[ \varepsilon(\omega,k)|E|^2 \right] \]

という 有効ラグランジアン が書ける。

(b) エネルギーの定義

時間並進に対応する Noether 量：

$$ W = \omega \frac{\partial \mathcal{L}}{\partial \omega}

  • \mathcal{L} $$

これを計算すると：

\[ W = \frac{|E|^2}{16\pi} \frac{\partial}{\partial\omega} (\omega\varepsilon) \]

👉 一発で出る そして

\[ \mathcal{N} = \frac{W}{\omega} \]

が wave action。

4️⃣ 「一次解を使って確認する」ならどうやる？

Naoki さんの案を 正しい形 に直すとこう。

手順

  1. 流体方程式を線形化
  2. (\delta n, \delta u, \delta E) を解く
  3. 全エネルギー密度を二次まで展開
  4. 時間平均
  5. 分散関係を代入して整理

例：電子プラズマ波

[ W = \frac{|E|^2}{8\pi}

  • \frac{1}{2}mn_0 |\delta u|^2 ]

に一次解

[ \delta u = \frac{eE}{m i\omega} ]

を代入すると：

[ W = \frac{|E|^2}{16\pi} \left( 1 + \frac{\omega_p^2}{\omega^2} \right) ]

これは

[ \varepsilon(\omega) = 1 - \frac{\omega_p^2}{\omega^2} ]

を使えば

[ \frac{\partial}{\partial\omega}(\omega\varepsilon) = 1 + \frac{\omega_p^2}{\omega^2} ]

と一致。

👉 確認としては OK 👉 定義としては NG

5️⃣ なぜ「保存則 → 定義」じゃないとダメか

  • ゲージ不変性
  • 一意性
  • 群速度との整合性
  • 波束・非一様系への拡張

これらは 「振幅²っぽいから」では保証されない。

まとめ（短く）

  • ❌ 一次摂動だけでは足りない
  • ⭕ 二次まで含めた エネルギー保存則 から導出する
  • ⭕ 変分原理を使うと本質が一番きれい
  • ⭕ 一次解は「検算」には使える

この先、

  • Vlasov 系でやるとどこに Landau 減衰が潜むか
  • 強減衰でこのエネルギー概念が壊れる瞬間
  • 「負のエネルギー波」

どれに行く？

その違和感は完全に正しいです。 結論から言うと、

wave action は「あとから定義された便利量」ではなく、 もともと“対称性＋変分原理”から必然的に出てくる保存量

で、それを示さずに使うのは本来ダメ、という感覚は理論的に健全です。

以下、「どこから来たか」だけを一直線に説明します。

0️⃣ 何を示せば「来歴を示した」ことになるか

最低限示すべきなのは：

  1. 波動の有効作用（ラグランジアン）が書ける
  2. そこに 位相の平行移動対称性 がある
  3. その Noether 量が wave action

これが示せて初めて

「action は保存する」 と言ってよくなります。

1️⃣ 出発点：線形波動は必ず二次作用を持つ

Vlasov–Maxwell でも、流体–Maxwell でも、

  • 平衡解のまわりで展開
  • 一次方程式は線形
  • 作用の二次部分だけを抜き出せる

すると必ず

\[ S^{(2)} = \int dt,d^3x; \mathcal{L}^{(2)} \]

が定義できる。

これは「仮定」じゃなくて：

線形方程式がある ⇔ 二次作用が存在

という事実。

2️⃣ WKB 展開で「位相」と「振幅」を分離する

波を

\[ \mathbf{E} = \mathbf{A}(\mathbf{x},t) e^{i\theta(\mathbf{x},t)} \]

と書く（幾何光学近似）。

  • (\theta)：速く変化（位相）
  • (\mathbf{A})：ゆっくり変化（包絡）

このとき

\[ \omega = -\partial_t \theta,\quad \mathbf{k} = \nabla \theta \]

3️⃣ 有効ラグランジアンの構造（ここが核心）

二次作用を代入して平均すると、

$$ \boxed{ \mathcal{L}_{\text{eff}} = \mathcal{N} \left( \partial_t \theta

  • \omega(\mathbf{k},\mathbf{x}) \right) } $$

という普遍形になる。

ここで

\[ \mathcal{N} = \frac{|E|^2}{16\pi} \frac{\partial \varepsilon}{\partial \omega} \]

（係数は系によるが構造は不変）

👉 これは「定義」ではなく WKB 極限で必ず出てくる形。

4️⃣ 位相の平行移動対称性

\[ \theta \rightarrow \theta + \text{const} \]
  • 物理は不変
  • 明らかな連続対称性

→ Noether の定理より保存量がある。

計算すると：

\[ \boxed{ \mathcal{N} = \frac{\partial \mathcal{L}_{\text{eff}}} {\partial(\partial_t \theta)} } \]

これが wave action density。

5️⃣ 保存則の形

Euler–Lagrange 方程式より：

$$ \boxed{ \partial_t \mathcal{N}

  • \nabla\cdot(\mathbf{v}_g \mathcal{N}) = 0 } $$

  • (\mathbf{v}_g = \partial \omega / \partial \mathbf{k})

  • 波束が「粒子」のように運ばれる

👉 ここで初めて 「action は断熱的に保存する」と言える。

6️⃣ エネルギーとの関係は“結果”

エネルギー密度は

\[ W = -\frac{\partial \mathcal{L}_{\text{eff}}}{\partial t} \]

背景が時間一定なら：

\[ \boxed{ W = \omega \mathcal{N} } \]
  • action が根源
  • エネルギーはその結果

これが重要な力関係。

7️⃣ なぜエネルギーより action が「本質」なのか

  • 背景が時間変化すると → エネルギーは保存しない
  • 位相対称性が壊れない限り → action は保存

だから：

波動の「粒子性」はエネルギーではなく action に宿る

8️⃣ Naoki さんの違和感を言語化すると

「energy/ω を action と呼ぶ」 という説明は 逆

正しくは：

位相対称性 → action → energy = ω action

まとめ（超要約）

  • wave action は「便利な量」ではない
  • 二次作用＋位相対称性から必然的に出る
  • Noether 量として一意に定まる

  • エネルギーより論理的に上流

  • Landau 減衰で action 保存がどう壊れるか

  • 負エネルギー波と action の符号
  • トカマク帯状流で「何が action なのか」

とても大事な疑問だし、ここが腑に落ちると wave action／エネルギーの話が全部一本につながるところです。 少し丁寧に、でも「なぜ必然なのか」だけに絞って説明します。

結論を先に

線形波動方程式が与えられているなら， それは必ず「ある二次作用の Euler–Lagrange 方程式」として書ける。

これは「特定のモデルではそうできる」ではなく、

自己随伴（self-adjoint）な線形演算子 → 必ず二次形式を持つ

という かなり一般的な数学的事実です。

1️⃣ まず最も単純な例から（直感）

単振動子

[ \ddot q + \omega^2 q = 0 ]

これは線形方程式。

対応する作用は：

[ S = \int dt; \frac{1}{2} \left( \dot q^2 - \omega^2 q^2 \right) ]

なぜ二次か？

  • 方程式が (q) に関して線形
  • 変分するときは ( \delta q ) が掛かる
  • → ラグランジアンは 最低でも (q^2)

ここでは誰も違和感を持たない。

2️⃣ 一般の線形波動方程式で何が起きているか

一般形で書く：

[ \hat L \psi = 0 ]

  • (\psi)：場（電場、密度揺らぎ、位相…）
  • (\hat L)：線形微分演算子

ここで重要なのは：

物理的な波動方程式の (\hat L) は 必ず自己随伴（エネルギー保存可能）

自己随伴って何？

[ \int d^4x; \phi, (\hat L \psi) ===================

\int d^4x; (\hat L \phi), \psi \quad (+\text{境界項}) ]

が成り立つこと。

これは

  • Maxwell
  • 流体
  • Vlasov（衝突なし）

すべてで成り立つ（減衰がなければ）。

3️⃣ 自己随伴演算子 → 二次作用（核心）

ここが一番重要。

作用をこう定義する：

[ \boxed{ S[\psi] = \frac{1}{2} \int d^4x; \psi, \hat L \psi } ]

これを変分すると：

[ \delta S = \int d^4x; \delta\psi, \hat L \psi \quad(+\text{境界項}) ]

よって：

[ \frac{\delta S}{\delta \psi} = \hat L \psi ]

つまり：

[ \boxed{ \delta S = 0 ;\Longleftrightarrow; \hat L \psi = 0 } ]

👉 線形方程式そのものが Euler–Lagrange 方程式

4️⃣ なぜ「必ず」二次なのか

ここで論理をはっきりさせると：

  1. 方程式は (\psi) に線形
  2. 変分は (\delta\psi) に線形
  3. だから作用は (\psi \times \psi)（二次）しかありえない

一次作用だと変分して定数しか出ない。 三次以上だと方程式が非線形になる。

👉 線形性 ⇔ 二次作用

5️⃣ プラズマ波動の場合はどうなるか

例：電場表示

線形応答を書くと：

[ D(\omega,\mathbf{k}) E = 0 ]

（(D = k^2 - \varepsilon(\omega) \omega^2 / c^2) など）

対応する作用は：

[ S^{(2)} = \frac{1}{16\pi} \int d^4x; E, D(i\partial_t,-i\nabla), E ]

これは

  • Maxwell の作用
  • 粒子の運動エネルギー

を 二次でまとめたもの。

6️⃣ 「でも Vlasov は非局所・複素だよね？」問題

良いところに気づいている。

Landau 減衰がない場合

  • (\varepsilon(\omega)) は実
  • (\hat L) は自己随伴
  • → 作用が書ける

Landau 減衰あり

  • (\varepsilon) が複素
  • (\hat L) は非自己随伴
  • → 厳密な作用は存在しない

👉 ここで初めて 「wave action 保存が壊れる」 という話につながる。

7️⃣ ここまでを一文で言うと

線形・非散逸な波動方程式は、 自己随伴演算子で書けるため、 それ自身を核とした二次作用を必ず持つ。

だから wave action は 「後付けの便利量」ではなく 作用の存在そのものから出てくる。

8️⃣ Naoki さんの理解段階を言語化すると

たぶん引っかかっていたのは：

「方程式を見ただけで どうして作用があると言えるの？」

という点。

答えは：

保存的な線形方程式である、という事実そのものが 作用の存在をほぼ保証している

もし次に進むなら：

  • Maxwell–Vlasov の二次作用を実際に書く
  • 自己随伴性が破れると何が起きるか（負エネルギー波）
  • 「帯状流」は action を持つのか？

どれも今の疑問の自然な延長だと思う。

うん、その式は基本的に正しいです。ただし、成立条件と意味づけをかなり厳密に押さえておく必要があります。 Naoki さんが気にしている「波のエネルギー容量はどれくらいか」「流れの運動エネルギーとどう切り分けるか」という文脈に、まさに直結する式です。

結論（まず要点）

線形・弱減衰のプラズマ波に対して、 単色平面波 ( (\omega,\mathbf{k}) ) について

[ \boxed{ W_w = \frac{|E|^2}{16\pi}, \frac{\partial}{\partial\omega} \bigl[\omega,\varepsilon(\omega,\mathbf{k})\bigr] } ]

という形で 波動エネルギー密度（正確には 平均エネルギー密度）が書けます。

これは

  • 電磁場のエネルギー
  • 粒子の秩序運動エネルギー（振動エネルギー）

を まとめた量 です。

どこから来る式か（物理的意味）

この式は

  • 分散媒質中の電磁エネルギー保存則
  • または Poynting 定理を分散媒質に拡張したもの
  • Brillouin, Landau–Lifshitz, Stix などの古典的結果

に基づきます。

ポイントは：

プラズマ中では 「電場エネルギー = ( |E|^2/8\pi )」 だけでは 全エネルギーを数え切れない

ということ。

粒子が電場に引きずられて振動しており、その運動エネルギーが (\varepsilon(\omega,k)) の (\omega)-依存性として隠れている。

それを正しく拾うのが

[ \frac{\partial}{\partial\omega}(\omega\varepsilon) ]

という因子。

係数 (16\pi) について

これは

  • 実効値（時間平均）
  • 複素振幅表記
  • ガウス単位系

を使った結果です。

例えば [ \mathbf{E} = \Re{ \mathbf{E}_0 e^{-i\omega t}} ] なら

[ \langle |E|^2\rangle = \frac{1}{2}|E_0|^2 ]

が効いて、最終的に (16\pi) が出ます。 文献によって (8\pi) が出てきても、定義の違いなので注意。

どういう場合にダメか

この式は万能ではありません。

❌ 成立しない／危ないケース

  • 強い減衰（(\Im \varepsilon \not\ll \Re \varepsilon)）
  • 非線形波動
  • 波が粒子を捕捉している場合
  • 強い空間非一様性
  • 波束の急激な時間変化

特に Landau 減衰が強いときは、 「波のエネルギー密度」という概念自体が曖昧になります。

wave action との関係（Naoki さんの関心点）

このエネルギー密度から

[ \boxed{ \mathcal{N} = \frac{W_w}{\omega} } ]

を定義したものが wave action。

  • 断熱変化に対して保存
  • 背景がゆっくり変わるときに有効
  • 幾何光学近似の本体

なので、

「エネルギーは保存しないが、action は保存する」

というあの話につながります。

比熱との比較という観点で言うと

Naoki さんの直感はほぼ正しくて、

  • 通常の線形波動では (W_w \ll nT)
  • 大きくなると それはもう「流れ」や「非線形運動」

として扱うべき領域に入る。

この式は 「波として数えられる最大のエネルギー」 を与えているとも言えます。
