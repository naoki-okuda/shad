# CGS-ガウス単位系（Gaussian-cgs units）

前半部「CGS」と後半部「ガウス」でそれぞれ以下を示している。
- CGS単位系 : Centimetre–gram–second system of unitsのこと。基本単位として、cm/g/sを選択する。
- ガウス単位系：非有理化単位系のこと。マクスウェル方程式で$4\pi$の係数が残るように、電気量を正規化した単位系。SIから見ると、$4\pi \epsilon_0 = 1$を課したものに相当する。

CGS単位系はプラズマ物理学では現役で使われる。(wikipediaにはもはやCGS-Gaussは使われないような記述のされ方をしているが、そんなことはない。ただ、正直heaviside-lorentzの方が洗練されていると思う。。)


# 磁場

磁場の単位にはしばしば、Tesla, Gaussが用いられる。これらは組み立て単位として以下のように与えられる。


$$ 
\mathrm{Tesla = N / (A m ) } 
$$

$$
\mathrm{Gauss = \sqrt{dyne}/cm = \sqrt{10^{-5}N}*10^2/m}
$$

Gaussが磁場の単位であるにもかかわらず電荷の次元を含まないのは、
CGS単位系の習慣として、SI基本単位A,kg,m,s間に次のような束縛を導入するからだ。

$$ 
    4\pi \epsilon_0 = 1
$$ 

この束縛から、光速分を括り出してAについて解くと、

$$ 
 4\pi = c^2 \mu = c^2 4\pi* 10^{-7} \mathrm{N/A^2 }
$$

$$
\mathrm{A} = c \sqrt{ \mathrm{10^{-7} N} }
$$

Teslaに代入して、CGS準拠で、cm, g, sベースで書き直すと、

$$

# 物理定数の無次元化とCGS単位系の習慣


* 物理定数を「適当な実数（例えば 1 や (\pi)）」に置き換えても計算上は困らない。
  光速 (c=1)、プランク定数 (\hbar = 1) といった自然単位系が典型例。

  ただし、無次元化できる物理定数の個数には限りがある。
  基本次元が (L, T, M, Q) の 4 種類なら、独立な物理定数は最大4つまで任意の数にできる。
  例えば、電気定数($\epsilon_0$)、光速 (c)、プランク定数 (\hbar)、重力定数 (G) を1にすると、電荷・長さ・時間・質量の間に4本の関係式が入り、すべてが無次元化される。
  ここにさらに「フェルミ定数 (G_F) も1にする」といった条件を追加すると、独立性を超えるため矛盾が生じる。

* 逆に、物理定数を「増やす」こともできる。
  SI 単位系は m, kg, s, A の4つを基本とし、他の量は組立単位にしているが、もし別の組立量を基本量に昇格させれば、その変換係数が新たな物理定数として扱われる。
  これは光速を1に置く操作の逆である。
  
  現在の SI では光速を定義値として固定することで、長さと時間の間に 1 つの換算係数（$299792458 \, \mathrm{m/s} = c $ ）を導入している。
  
  光速を1とする立場は、そもそも長さと時間を区別する必然性が弱いという態度といえる。

* CGS 単位系では、真空の誘電率（あるいは透磁率）を適当な実数として定義する習慣がある。
  典型的には
  [
  4\pi\epsilon_0 = 1
  ]
  とおく。
  この操作により電荷は独立した基本量ではなくなり、長さ・時間・質量から組み立てられる量に“転落”する。
  （もちろん電荷を基本単位に残すこともできるが、CGS では通常 L・M・T の3つを基本量として採用する。）

* この設定のもとでは、Maxwell 方程式が (E) と (B) の間でより対称的な形になる。
  さらに光速 (c=1) とすると、式はもっと簡潔になる。
  (\epsilon_0=1) とする流儀も存在する。

# 微細構造定数 α は1にできない量

理由はシンプルで、**α は自然界の相互作用強度そのものであり、単位系には依存しない“無次元の物理定数”だから**。

[
\alpha = \frac{e^2}{4\pi\varepsilon_0 \hbar c}
]

単位系で

* (c), (\hbar), (\varepsilon_0), (e)
  をどう再定義しても、
  **比率としての α の値は固定される**。

実際の値は

[
\alpha \approx \frac{1}{137.035999084\dots}
]

で、これは変えられない「自然の設定値」。

# ✔ 「無次元定数」は単位系で変えようがない

物理定数には2種類ある：

## ●（1）単位系で値が変えられる定数

* c（光速）
* ħ（プランク定数）
* G（重力定数）
  これらは “次元” を持つので、単位を変えれば値も変えられる。

## ●（2）単位系では変えられない定数

* α（微細構造定数）
* 各種 Yukawa coupling
* CKM の角度
* ニュートリノ混合角
  これらは無次元なので、**単位の選び方で値は絶対に変わらない**。

まさに α はこの典型例。
