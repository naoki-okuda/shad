本記事は正確な記述を欠いている可能性が高いと認識しています。 なぜなら、単位系周りの話は、習慣的用法が入り混じる世界で、定義など時代によって異なるからです。

プラズマの文脈では標準的にCGS-ガウス単位系が使われます。多くの文献・数値計算コードがCGS-ガウス単位系に準拠しています。

単位等の換算について、時々無意味に混乱することがあるために自分の脳内を整理した備忘録となります。

CGS-ガウス単位系（Gaussian-cgs units）

前半部「CGS」と後半部「ガウス」でそれぞれ以下を示している。 - CGS単位系 : Centimetre–gram–second system of unitsのこと。基本単位として、cm/g/sを選択する。 - ガウス単位系：非有理化単位系のこと。マクスウェル方程式で\(4\pi\)の係数が残るように、電気量を正規化した単位系。SIから見ると、\(4\pi \epsilon_0 = 1\)を課したものに相当する。

CGS単位系はプラズマ物理学では現役で使われる。(wikipediaにはもはやCGS-Gaussは使われないような記述のされ方をしているが、そんなことはない。ただ、正直heaviside-lorentzの方が洗練されていると思う。。)

磁場

磁場の単位にはしばしば、Tesla, Gaussが用いられる。これらは組み立て単位として以下のように与えられる。

\[ \mathrm{Tesla = N / (A m ) } \]
\[ \mathrm{Gauss = \sqrt{dyne}/cm = \sqrt{10^{-5}N}*10^2/m} \]

Gaussが磁場の単位であるにもかかわらず電荷の次元を含まないのは、 CGS単位系の習慣として、SI基本単位A,kg,m,s間に次のような束縛を導入するからだ。

\[ 4\pi \epsilon_0 = 1 \]

この束縛から、光速分を括り出してAについて解くと、

\[ 4\pi = c^2 \mu = c^2 4\pi* 10^{-7} \mathrm{N/A^2 } \]
\[ \mathrm{A} = c \sqrt{ \mathrm{10^{-7} N} } \]

Teslaに代入して、CGS準拠で、cm, g, sベースで書き直すと、

\[ \mathrm{Tesla = N/ A m} = \mathrm{N} /(c \sqrt{10^{-7} \mathrm{N} } \mathrm{m} ) = 10^{-2}/\mathrm{cm}/c \sqrt{10^{12}\mathrm{dyne} } = 10^4 \mathrm{Gauss}/c \]

物理定数の無次元化とCGS単位系の習慣

  • 物理定数を「適当な実数（例えば 1 や (\pi)）」に置き換えても計算上は困らない。 光速 (c=1)、プランク定数 (\hbar = 1) といった自然単位系が典型例。

ただし、無次元化できる物理定数の個数には限りがある。 基本次元が (L, T, M, Q) の 4 種類なら、独立な物理定数は最大4つまで任意の数にできる。 例えば、電気定数(\(\epsilon_0\))、光速 (c)、プランク定数 (\hbar)、重力定数 (G) を1にすると、電荷・長さ・時間・質量の間に4本の関係式が入り、すべてが無次元化される。 ここにさらに「フェルミ定数 (G_F) も1にする」といった条件を追加すると、独立性を超えるため矛盾が生じる。

  • 逆に、物理定数を「増やす」こともできる。 SI 単位系は m, kg, s, A の4つを基本とし、他の量は組立単位にしているが、もし別の組立量を基本量に昇格させれば、その変換係数が新たな物理定数として扱われる。 これは光速を1に置く操作の逆である。

現在の SI では光速を定義値として固定することで、長さと時間の間に 1 つの換算係数（$299792458 \, \mathrm{m/s} = c $ ）を導入している。

光速を1とする立場は、そもそも長さと時間を区別する必然性が弱いという態度といえる。

  • CGS 単位系では、真空の誘電率（あるいは透磁率）を適当な実数として定義する習慣がある。 典型的には [ 4\pi\epsilon_0 = 1 ] とおく。 この操作により電荷は独立した基本量ではなくなり、長さ・時間・質量から組み立てられる量に“転落”する。 （もちろん電荷を基本単位に残すこともできるが、CGS では通常 L・M・T の3つを基本量として採用する。）

  • この設定のもとでは、Maxwell 方程式が (E) と (B) の間でより対称的な形になる。 さらに光速 (c=1) とすると、式はもっと簡潔になる。 (\epsilon_0=1) とする流儀も存在する。

  • 真空の誘電率は物理定数である。$4\pi \epsilon_0 $

  • 物理定数を適当な数(基本実数だろう。例えばpiとか)に置き換えても、計算上特段困りはしない。例えば、しばしば光速を1、プランク定数を1に置き換えることがある。 ただし、このような操作によって無次元化できる独立な物理定数の数には限りがある。例えば、電気素量と光速とプランク定数と重力定数を適当な数だと見做した場合、電荷と長さと時間と質量の間に４本の束縛関係式が付くわけなので、これらの量が無次元の数で表せることになる。これらに加えて、例えば、「フェルミ定数[エネルギーのマイナス二乗]も1に」という関係式を追加すると矛盾してしまうので困る。

  • 物理定数を消す逆操作として物理定数を無為に増やしていくことは可能だと言える。SIの単位系ではm/kg/s/Aを基本の単位として他の物理量は組み立て単位であるとする。とある組み立て単位を基本単位に取り入れて、その換算関係式の係数を新たな物理定数にしてしまえば、「物理定数」が増えたことになる。これは光速を１におく操作の逆である。実際にSI単位系の定義において、光速を定義値c = 2.99792458 * 10^8 m/sとして、余計に適当な実数299792458を導入することで長さと時間を別の基本単位に変えてしまっている。(光速を1とする立場はそもそも長さと時間は区別される必要のない量であるという態度の表明だ。)

  • CGS単位系の習慣では、真空の誘電率(あるいは真空の透磁率)を実数に置き換える。この操作によって、電荷の単位は長さと時間と質量の組み立て単位に転落する。（電荷を基本的な単位に連れてきても良いが、習慣的に長さと質量と時間を残す。）

    4pi \epsilon_0 = 1
  • この操作によって、maxwell方程式の記述がE,Bに関して対称的になる(光速を１にするともっと簡単になる)。\(\epsilon_0 = 1\)とする宗派もある。

微細構造定数 α は1にできない量

理由はシンプルで、α は自然界の相互作用強度そのものであり、単位系には依存しない“無次元の物理定数”だから。

[ \alpha = \frac{e^2}{4\pi\varepsilon_0 \hbar c} ]

単位系で

  • (c), (\hbar), (\varepsilon_0), (e) をどう再定義しても、 比率としての α の値は固定される。

実際の値は

[ \alpha \approx \frac{1}{137.035999084\dots} ]

で、これは変えられない「自然の設定値」。

✔ 「無次元定数」は単位系で変えようがない

物理定数には2種類ある：

●（1）単位系で値が変えられる定数

  • c（光速）
  • ħ（プランク定数）
  • G（重力定数） これらは “次元” を持つので、単位を変えれば値も変えられる。

●（2）単位系では変えられない定数

  • α（微細構造定数）
  • 各種 Yukawa coupling
  • CKM の角度
  • ニュートリノ混合角 これらは無次元なので、単位の選び方で値は絶対に変わらない。

まさに α はこの典型例。

✔ α=1 にしたいという願望は QED を完全に別の理論にしてしまう

α が 1 に近づいた世界では

  • 水素原子は存在しない
  • 電子は原子核に束縛されず、原子のスペクトルも消える
  • 電磁相互作用は潮汐効果的に強すぎて場の不安定性が起きる といった “別の宇宙” になる。
